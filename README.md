# gribberish

This branch includes major updates to the infrastructure - including eccodes support and ensembles, along with following naming conventions similar to cfgrib. See the [docs](docs/) directory for more specific instructions.

Read [GRIB 2](https://en.wikipedia.org/wiki/GRIB) files with Rust.

See [`gribberish`](gribberish/README.md) for the core library

See [`python`](python/README.md) for usage with `python` and `xarray`

See [`node`](node/README.md) for usage with `nodejs`
